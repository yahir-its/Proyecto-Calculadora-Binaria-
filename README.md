# Calculadora Binaria de 2 Bits
Este proyecto consiste en el diseño y desarrollo de una calculadora binaria capaz de realizar operaciones básicas utilizando únicamente números binarios de 2 bits.
Incluye la parte física (circuito con compuertas lógicas), el programa en Java y la documentación generada durante el proyecto.

# Funciones de la calculadora
- La calculadora puede realizar:
- Suma
 -Resta
- Multiplicación
- División
- Trabajando siempre con valores binarios representados con 2 bits.

  # Programa en Java
Se desarrolló un programa en Java que permite realizar las mismas operaciones binaras que el circuito físico.
Este programa fue elaborado en NetBeans y ayuda a comprender mejor el funcionamiento de la aritmética binaria.

Incluye:
- Validación de datos
- Conversión binaria
- Ejemplos de corridas
